# x86

A tiny x86 emulator written in Rust.

## Requirements

- gcc
- ld
- nasm
- rust

## Build

```bash
make
cargo build
```

## Usage

```bash
cargo run path/to/binary
# e.g.
cargo run samples/helloworld/helloworld.bin
```
